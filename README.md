# Fundamentos
Repositório criado para armazenar exercícios, exemplos e práticas do livro Fundamentos da Programação de Computadores. ASCENCIO, Ana Fernanda Gomes; CAMPOS, Edilene Aparecida Veneruchi de. Fundamentos da programação de computadores: algoritmos, PASCAL, C/C++ (padrão ANSI) e JAVA. 3. ed. São Paulo: Pearson, 2012. E-book.
